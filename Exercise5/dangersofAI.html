Oι κίνδυνοι της Τεχνητής Νοημοσύνης

Εκτός από τις Εφαρμογές της τεχνητής νοημοσύνης και τα θετικά που μπορεί να προσφέρει σε μία κοινωνία ,δεν ξεχνάμε πως υπάρχουν και τα αρνητικά που μπορεί να επιφέρει.

Οικονομικό κόστος

 Τα τεχνητά μηχανήματα μπορεί να κοστίσουν αρκετά χρήματα και χρόνο για να δημιουργηθούν,να διορθωθούν και να ανανεωθούν.Η ρομποτική επιδιόρθωση θα μπορούσε να βοήθησει να μειωθεί ο χρόνος, αλλά αυτό θα κόστιζε παραπάνω λεφτά και θα χρειαζόντουσαν πόροι.

Αύξηση της ανεργίας

 Τα μηχανήματα θα μπορούνε να αντικαταστήσουν δουλειές και έτσι θα οδηγήσουν σε ανεργία ένα μεγάλο μέρος του πληθυσμού. Οι κυβερνήσεις πρέπει να λάβουν τα απαραίτητα μέτρα ώστε να μην μεγαλώσει η ανεργία και οι άνθρωποι που έχασας τις δουλειές τους λόγω των ρομπότ να εξειδικευθούν σε άλλο τομέα, ώστε η τεχνητή νοημοσύνη να προσφέρει τα ευεργετικά της αποτελέσματα στην κοινωνία.

Καθόλου συναισθήματα

 Δεν υπάρχει αμφιβολία πως τα μηχανήματα είναι πολύ καλύτερα όταν πρόκειται να δουλέψουν αποδοτικά αλλά σίγουρα δεν μπορούν να αντικαταστήσουν ανθρώπινη σύνδεση που τους κάνει ομάδα. Οι μηχανές δεν μπορούν να δεθούν με τους ανθρώπους το οποίο είναι ένας σημαντικός παράγοντας όταν πρέπει να συνεργαστούν σαν ομάδα.

 Ωστόσο, η τεχνητή νοημοσύνη έχει θετικές και αρνητικές επιπτώσεις. Ένα βασικό μέλημα που δεν πρέπει να ξεχνούν οι άνθρωποι είναι πως η άνοδος των ρομπότ δεν πρέπει να βρεθεί εκτός ελέγχου.
